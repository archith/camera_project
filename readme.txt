============ Packages List ================================================
source/source		Firmware source code
source/toolchain	Toolchain source code

============ source code layout ==========================================
Source Structures:
	source/source/setenv		Script to set the environment varialbes
	source/source/build/		scripts to build FW
	source/source/host/		PC Host relative files (scripts, mkcramfs...)
	source/source/img/		Temporary image files (kernel, filesystem...) which
					are generated by run-time
	source/source/lib/		libraries for the F/W (default libraries)
			modules/	kernel modules (by run-time)
	source/source/src/		All source files
	   		bootloader/	boot loader
	   		kernel/		linux kernel
	   		kmods/		kernel modules
	   		rootfs/		root filesystem (basic)
	   		sysapps/	applications
	source/source/FW/		Folder for building target FW
	source/toolchain/		toolchain

============ Build Steps ==================================================
1. Set environment
	1) cd source/source
	2) source setenv WVC2300 USA
2. Compile toolchain and Firmware source code
	1) make
3. Firmware location
	source/source/FW/WVC2300.bin

Attention: Only verified in OS Fedora 9
