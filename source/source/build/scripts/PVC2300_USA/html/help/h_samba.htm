SMB/CIFS

Check the check box to enable SMB/CIFS recording. SMB/CIFS allows the camera to record directly to the Network Attached Storage (NAS).

  • SMB/CIFS Server—Enter the server hostname or IP address.
  • Upload Path—Enter the folder name to upload video to SMB server.
  • Login Name—Enter user’s login name for SMB server.
  • Password—Enter user’s password for SMB server.
  • Maximum Size of Each File—Enter the maximum file size to upload to the SMB server.
  • Enable adding timestamp to files—When enabled, a timestamp is added to each recorded file and there will be multiple recorded files. Older files will not be over-written, so the size of your recording will continuously grow.
