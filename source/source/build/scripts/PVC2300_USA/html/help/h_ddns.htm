Help - DDNS

Use this feature to access the PVC2300 camera from a remote location.

DDNS
Enable DDNS Enables Dynamic Domain Name Service (DDNS). When used with a DDNS service provider, you can access your camera over the Internet using a domain name instead of an IP address. From the drop-dwon list, choose DynDNS, TZO, 3322.
Service Provider Select the service provider that you have an account with. If you do not have an account, you can select a service provider and click Web Site.
Host Name Enter the host name (Domain Name) allocated to you by the DDNS service provider.
Account Enter the login name for the DDNS account.
Password Enter the password for the DDNS account.
Check WAN IP Address Allows you to choose how often the device should check to see if the IP address has changed. If the IP address has changed, the DDNS server will be notified.
Starting At Use the “Starting At” to set the schedule for checking if the WAN IP address has changed.
Note: To complete the DDNS Service setup, port forwarding needs to be configured on your router.
