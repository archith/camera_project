Help - System

System Status
Firmware version The version of the firmware currently installed is displayed.
IP address Displays the IP address for camera.
MAC Address Displays the current MAC address of the camera.
Camera Name Displays the name of the camera, as defined on the Basic > Device Settings window. By default, the last six digits of the camera name come from the last three bytes of the MAC address.
Description Displays the description of the camera, as defined on the Basic > Device Settings window.
Date/Time Displays the current date/time configured on the camera.
