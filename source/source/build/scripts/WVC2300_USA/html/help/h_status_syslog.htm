Help - Syslog & Log

Log Type
Log Type Check the check box of the types of logs that you want to view on the Log List.

Syslog Server
Enable Syslog Server Check the box to enable the Syslog server feature.
Syslog Server Address Enter the address of your Syslog server.
Syslog Port To change, enter the Syslog port number (1024 to 65535) in the field provided.   The default Syslog port is 514.

Log List
Log List This displays system activities. Click Clear Log to erase all the entries in the System Log.
