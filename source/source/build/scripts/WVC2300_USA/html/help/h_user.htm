Help - Users

Administrator
Admin

The default administrator password should be changed for security purposes. To change the administrator password, enter the desired password in the Password and Confirm Password fields. Click Change to apply the change.

Note: Spaces, punctuation, and special characters are not supported. The name is case sensitive. The default administrator name (admin) cannot be changed.

Demo
Viewer This account allows the viewing of video only. By default, the account name is demo and there is no password assigned. To change the demo account settings, modify the user name and password in the appropriate fields and click Change.

Privilege Level

 Select the appropriate privilege level from the drop-down list. Choose from one of the following options:

  • Administrator—Provides the specified user full camera administration and control privileges.

  • Monitor—Provides the specified user control of the camera video (manually pan/tilt, toggle between day/night vision, and trigger output ports).

  • Viewer—Provides the specified user video viewing privileges only.
Add and Delete Click Add to add a new user ID or click Delete to remove an existing user ID. Click Save to save the user configuration.
