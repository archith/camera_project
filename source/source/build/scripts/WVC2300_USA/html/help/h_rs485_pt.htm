Help - Set Pan/Tilt
Set Positions Set the desired position through adjusting the control panel.
Set Home Button Set the desired position and then click this button to set Home position.
Remove Preset List Select the preset position you don't need and then click the "Delete" button to remove.
Preset Name Name the current desired position and then click the "Add" button to add into the preset list.
Pan Speed Set the Tilt direction speed
Tilt Speed Set the Pan direction speed
