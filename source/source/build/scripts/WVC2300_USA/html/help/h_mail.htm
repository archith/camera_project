SMTP Server (Primary & Secondary)

 SMTP Mail Server (Primary and Secondary)
  • SMTP Mail Server—Enter the address of the SMTP (Simple Mail Transport Protocol) Server to be used to send e-mail.
  • Port—Enter the port number for the mail server. The default is 25.

Authentication—From the drop-down list, choose one of the following options depending on what form of authentication your SMTP server supports:

  • Not Required—Authentication is not required to send mail.
  • Requires SMTP Authentication—Select if the SMTP server requires a login to send mail.

    • - Account Name—Enter the account name to be used on the e-mail server.
    - Password — Enter the password associated with the account name.

  • Requires POP before SMTP—Select if the SMTP server requires a POP login to send mail. Check the box if your mail server requires authentication and enter the following options:

    • - Account Name—Enter the account name to be used on the email server.
    - Password—Enter the password associated with the account name.
    - POP Server Name—Enter the hostname or IP address of the POP3 mail server that is being used to receive e-mail.
Test SMTP Server Click to test the configuration. You will get an error message if the transaction does not complete. For example, test fails due to a wrong username or password.
Requires SSL Encryption Check the check box if your mail server requires SSL encryption. This option can support cryptographic protocols that provide security and data integrity for communications over TCP/IP networks such as the Internet.
Enable Secondary SMTP

Check the box to upload to the secondary SMTP server if the camera cannot connect to the primary SMTP server.

Note: Only one SMTP server is used, even if both the primary and secondary SMTP servers are enabled. The secondary SMTP server is used if the camera cannot connect to the primary server.

E-mail Setup
E-Mail Setup
  • Send To—Enter at least one (1) e-mail address; the 2nd and 3rd addresses are optional. The e-mail alert will be sent to the e-mail address or addresses specified here.
  • Show “From” as—Enter the e-mail address to be shown in the From field when the e-mail is received.
  • Subject—Enter the desired text to be shown as the subject for the e-mail when it is received. Maximum field length is 48 characters.
