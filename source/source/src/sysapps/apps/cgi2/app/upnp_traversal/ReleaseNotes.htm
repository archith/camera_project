Overview

These release notes outline the new components and main functions of Intel® Development Toolkit for Intel® NMPR V 2.0. Additional information regarding technical support, system requirements and known limitations is also included.

Product Contents

The current release of Intel® Development Toolkit for Intel® NMPR v2.0 includes these main modules:

  • Intel® Digital Media Server Simulator

  • Intel® Remote UI Application Simulator

  • Intel® Device Builder

  • Intel® Device Validator

  • Intel® Device Monitor

  • Intel® Service Author

  • Digital Media Server Microstack

  • Digital Media Player Microstack

  • Remote UI Client Microstack

Tools

The following tools are included in this release:

  • Intel® Digital Media Server (DMS) Simulator - Enabling testing of a DMP by providing different kinds of test cases. These test cases, such as the 256 bytes ClassID, 1024 bytes item title etc. are available from the DMS Simulator user interface.

  • Intel® Remote UI Application Simulator - Enabling testing of a Remote UI Client by providing different kinds of tests for remotable Applications. You can run your remote application which connects to the Remote Application Simulator to test the features of your Remote UI Client.

  • Intel® Device Builder - The code generation tool enables you to generate a custom UPnP* device or control point microstack, conforming to NMPR v2.0 protocols.

  • Intel® Device Validator - The validation tool for testing and validating UPnP* devices. This tool provides a full suite of tests for basic UPnP behaviors and a set of AV renderer tests for testing an AV device and the AV Content Directory.

  • Intel® Device Monitor - The UPnP* control point for detecting UPnP* devices and displaying detailed information about a device. You can also use the tool to invoke actions, monitor events and packets, and handle errors.

  • Intel® Service Author - Use this tool to create and edit Service Control Point Declaration (SCPD) XML files for UPnP* services.
     

Micorstacks

  • Digital Media Server Microstack - C language reference source code that can be used to quickly build UPnP* DMS microstacks. This package includes UPnP* code samples for Windows* systems which can be used as reference, instead of using a generic UPnP* microstack, or building one yourself.

  • Digital Media Player Microstack - C language reference source code that can be used to build DMP devices and control points based on UPnP* technologies quickly. This package includes UPnP* code samples for Windows which can be used as reference instead of using a generic UPnP* microstack, or building one yourself.

  • Remote UI Client Microstack - The Intel® Remote UI Client Microtack provides a sample implementation of Universal Plug and Play* (UPnP*) Remote UI Client device. It uses the Extended Remote Technology (XRT*) 2.2 as the remoting protocol. Use these codes as a starting point to develop your own Remote UI Client device, add a Remote UI feature to an existing device, or port it to other platforms.

New Features

As the upgrade of Intel® Development Toolkit for UPnP* Technologies, the new release provides the following new features below:

  • NMPR v2.0 conformance
    All components in this package have been, where possible, measured conformant to the appropriate NMPR v2.0 requirements.

  • Digital Media Server (DMS) and Digital Media Player (DMP) components
    The Control Point functionality has been merged into DMP component. In this release, there are only DMS and DMP microstacks. The Control Point is not an individual component any more.

  • Remote UI Application Simulator and Remote UI Client Software
    The Remote UI Application Simulator and Client provide more effective test application and reference code,     instead of Remote IO component in previous version.

  • Simulators
    There are two simulators be provided: DMS simulator and Remote UI Application simulator. These simulators enable you to test the client applications functionality.

System Requirements

The product modules have been validated in Windows* 2000/XP operating system.

For other operating systems, the validation is ongoing with the product development, please check out the Intel UPnP* related website for the updating information. See http://www.intel.com/technology/dhdevnet/.

Software requirements:

Developing Intel® NMPR v2.0 requires one of the following tools

  • Microsoft* Windows* 2000 professional with Service Pack 4, or Microsoft* Windows XP professional with Service Pack 1

  • Microsoft* .NET Framework 1.1

Developing Intel® NMPR v2.0 micorstacks requires one of the following tools

  • Microsoft* Windows 2000, or Microsoft* Windows XP professional with Service Pack 1, Red Hat* Linux* 9.0 (RUI Client, DMS only)

  • Microsoft* Visual C++ 6.0

  • Microsoft* Visual Studio* .NET 2003

Hardware requirements:

Processor:

  • Intel® Pentium II 450 MHz or above (Intel® Pentium III 600MHz recommended)

Memory:

  • Windows* 2000: 192 MB or above (256 MB recommended)

  • Windows* XP: 160 MB or above (192 MB recommended)

Free hard disk space:

  • Windows: System partition 600 MB or above, installation partition 3 GB or above

  • Red Hat Linux: follow the Linux default settings

Known Limitations

  • Linux platform support
    Remote UI Client and Digital Media Server stacks for Red Hat Linux 9.0 are provided with preview version features in this release;

  • Intel® Remote UI Client
     Choosing Digital Home(Fork)/(None-Fork) item from Connection menu in Intel® Remote UI Client, there may be a failure of UI initialization: the buttons in the UI can not be displayed;

  • Intel® Remote UI Client
     After a long time (more than 1 hour at least) running with Intel® Remote UI Application Simulator, sometimes Intel® Remote UI Client may get a failure which may cause a crash;

  • Digital Media Player Microstack
         In this release, Digital Media Player component (Windows Media Player v6.4 integrated) only support the media file format which be available in Windows Media Player v6.4.
